# 환경 구성
`Vite` + `Vue` 

# 🍕 프로젝트 생성
- https://vite.dev/guide/#scaffolding-your-first-vite-project

```shell
yarn create vite
```

### 설치 후 확인

```shell
yarn install
yarn dev
```

# 🍔 ESLint
- https://eslint.org/docs/latest/use/getting-started#quick-start

```shell
yarn create @eslint/config
```

### 설치 후 확인

```shell
yarn eslint .
```
